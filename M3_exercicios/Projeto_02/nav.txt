Esses exercícios são para praticar o que foi visto no conteúdo da aula sobre
Fundamentos da Web - Bootstrap. O exercício a seguir também é o projeto da semana,
utilize a aula para começá-lo e tirar suas dúvidas (não vale nota).
Descrição:
Os professores do troptech estão com dificuldade de cadastrar as notas dos alunos em
um local centralizado, por isso, vamos criar o front-end de um sistema para realizar a
gestão dessas notas.
Requisitos não-funcionais:
● Utilize Bootstrap;
● Utilize os aprendizados de css se necessário;
● Utilize o grid-system do Bootstrap;
● Todas as páginas devem possuir uma navbar para navegação entre as páginas
principais;
Requisitos funcionais:
● Ao acessar o sistema a página inicial deve ser uma mensagem de boas-vindas
explicando brevemente para o que se propõe o sistema, como inspiração deixo
esse exemplo: https://getbootstrap.com/docs/5.2/examples/starter-template/
● Um dos menus deve redirecionar para a tela de boas vindas;
● Um dos menus deve ser chamado de Alunos, ao clicar nele deve abrir uma tabela
com as informações: Aluno, Nota 1, Nota 2, Nota 3, Nota Final.
● Acima da tabela deve existir um botão para cadastrar um aluno e suas notas e
outro botão para editar as notas de um aluno existente;
● A tela de cadastro deve conter um formulário para inserção do nome do aluno e
suas três notas, a nota final será calculada automaticamente no futuro. O nome é
obrigatório, demais campos não;
● A tela de edição deve conter o nome do aluno em texto que está sendo editado e
os campos: Nota 1, Nota 2 e Nota 3 para edição (é apenas um exemplo, o nome do
aluno pode ser fixo na tela);
● Em todas as telas a navbar deve estar aparecendo;
